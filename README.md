# Finance Tracker

This is a simple finance tracker app I built using React to practice real-world concepts like CRUD operations, state management, and data calculations.

The app lets users add income and expenses, track balance, and see where money is being spent using a chart.

## Features

* Add income and expense transactions
* Edit and delete transactions
* See total balance, income, and expenses
* Categorize expenses (Food, Travel, etc.)
* Pie chart for expense categories
* Data saved in localStorage

## Tech Used

* React
* JavaScript
* CSS
* Recharts

## How to Run

bash
npm install
npm run dev
